Installation

ติดตั้ง Plugin

เพิ่มไม่กี่บรรทัดใน .claude/settings.json ของโปรเจคคุณ

1

เพิ่ม Marketplace ใน settings.json

{
  "extraKnownMarketplaces": {
    "superboom-marketplace": {
      "source": {
        "source": "github",
        "repo": "phiistphisit-glitch/superboom"
      }
    }
  },
  "enabledPlugins": {
    "superboom@superboom-marketplace": true
  }
}
2

Restart Claude Code

ปิดและเปิด Claude Code ใหม่ เพื่อให้ plugin โหลด
3

ทดสอบ

/quote นามบัตร Art Card 350gsm 4สี UV 1000 ชิ้น
