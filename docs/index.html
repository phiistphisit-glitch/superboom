Installation

ติดตั้งใน 3 ขั้นตอน

เพิ่ม config ไม่กี่บรรทัด แล้ว restart Claude Code

🔌 เพิ่ม plugin ใน settings.json

.claude/settings.json
1

เพิ่ม marketplace + enable plugin

เปิด .claude/settings.json ในโปรเจคของคุณ แล้วเพิ่ม:

{
  "extraKnownMarketplaces": {
    "superboom-marketplace": {
      "source": {
        "source": "github",
        "repo": "phiistphisit-glitch/superboom"
      }
    }
  },
  "enabledPlugins": {
    "superboom@superboom-marketplace": true
  }
}
2

Restart Claude Code

ปิดและเปิด Claude Code ใหม่ เพื่อให้ plugin โหลดจาก GitHub

3

ทดสอบ skill แรก

Claude Code

Claude Sonnet 4.6 ขับเคลื่อน superboom

superboom ใช้ claude-sonnet-4-6 เป็น backbone รองรับ Thai NLP และ multi-step reasoning สำหรับการคำนวณต้นทุนที่ต้องใช้ context ยาว

Agent SDK

Claude Code Plugin System เปิดให้ Community

Claude Code Agent SDK ช่วยให้สร้าง plugin ด้วย Markdown + YAML frontmatter ได้ทันที superboom เป็นตัวอย่างการนำ LLM มาใช้ใน SME
