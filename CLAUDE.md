# superboom — Thai Printing Shop Plugin

ปลั๊กอินสำหรับโรงพิมพ์ superboom ช่วยในการคำนวณราคา วิเคราะห์ต้นทุน
ตรวจสอบไฟล์งานพิมพ์ และเปรียบเทียบตัวเลือก รองรับออฟเซตและกราเวียร์

## คำสั่งที่ใช้ได้

| คำสั่ง | ฟังก์ชัน | ตัวอย่าง |
|--------|---------|---------|
| `/quote` | ใบเสนอราคา | `/quote นามบัตร Art Card 350gsm 4สี UV 1000 ชิ้น` |
| `/cost` | วิเคราะห์ต้นทุน | `/cost โบรชัวร์ A5 4/4 Laminate 2500 ชิ้น` |
| `/spec` | ตรวจ spec ไฟล์ | `/spec RGB 72dpi PDF ไม่มี bleed` |
| `/compare` | เปรียบเทียบตัวเลือก | `/compare 500 vs 1000 vs 5000 นามบัตร` |

## ประเภทงาน

- **Offset:** นามบัตร, โบรชัวร์, แผ่นพับ, ปฏิทิน, หนังสือ (500 ชิ้นขึ้นไป)
- **Gravure:** บรรจุภัณฑ์ฟิล์ม, ฉลากปริมาณสูง (10,000 ชิ้นขึ้นไป)

## ปัจจัยราคา

กระดาษ (Art Card/Bond/Craft) · สีพิมพ์ (1/2/4 สี CMYK) · Finishing (UV/Laminate/Die-cut/Emboss/Foil) · ค่าแม่พิมพ์ · Tiered quantity discount · ค่าขนส่ง · VAT 7%

## Agent

`print-specialist` — ผู้เชี่ยวชาญงานพิมพ์ รู้ราคาวัสดุ กระบวนการพิมพ์ และมาตรฐานไฟล์

ราคาเป็นการประมาณการ — ควร review ตารางราคาใน `agents/print-specialist.md` ให้ตรงกับต้นทุนจริงของโรงพิมพ์
