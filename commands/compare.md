# /compare — เปรียบเทียบตัวเลือกงานพิมพ์

เปรียบเทียบตัวเลือกงานพิมพ์ต่างๆ สำหรับโรงพิมพ์ superboom

**ตัวอย่างการใช้:**
- `/compare 500 vs 1000 vs 5000 ชิ้น นามบัตร Art Card 350gsm 4สี`
- `/compare offset vs gravure ฉลากสินค้า 4สี 20000 ชิ้น`
- `/compare art card 300gsm vs bond 100gsm โบรชัวร์ A5 4สี 1000 ชิ้น`
- `/compare UV coating vs laminate matt นามบัตร 1000 ชิ้น`

รับข้อมูลจาก: $ARGUMENTS

## ขั้นที่ 1: ระบุประเภทการเปรียบเทียบ

จาก $ARGUMENTS ระบุว่าต้องการเปรียบเทียบอะไร:

- **Quantity tiers**: มีตัวเลขจำนวนหลายค่า เช่น "500 vs 1000 vs 5000"
- **Offset vs Gravure**: มีคำ offset/gravure
- **Paper types**: มีชนิดกระดาษหลายชนิด
- **Finishing options**: มีรายการ finishing หลายอย่าง

ถ้า $ARGUMENTS ไม่ระบุชัดเจน ให้ถาม หรือทำ quantity comparison เป็น default
ถ้าไม่มี spec งาน (ขนาด, กระดาษ, สี) ให้ถาม

## ขั้นที่ 2: คำนวณทุกตัวเลือก

ใช้สูตรราคาเดียวกับ /quote สำหรับแต่ละตัวเลือก

**Tiered Quantity Comparison:**
คำนวณสำหรับแต่ละจำนวน (ตามที่ระบุ หรือ default: 500, 1,000, 2,500, 5,000, 10,000)
แสดง: ราคารวม, ราคา/ชิ้น, ประหยัด% เทียบกับจำนวนน้อยสุด

**Offset vs Gravure:**
คำนวณด้วย spec เดียวกัน ทั้ง 2 ประเภท
Breakeven = (Gravure Setup - Offset Setup) ÷ (ต้นทุนพิมพ์ต่อชิ้น Offset - ต้นทุนพิมพ์ต่อชิ้น Gravure)
แสดง: ราคา, คุณภาพ, lead time, จำนวน breakeven

**Paper type comparison:**
คำนวณต้นทุนกระดาษและราคารวมสำหรับแต่ละชนิด (spec อื่นเหมือนกัน)

**Finishing comparison:**
คำนวณราคารวมสำหรับแต่ละ finishing option (spec อื่นเหมือนกัน)

## ขั้นที่ 4: คำแนะนำ

หลังตารางให้วิเคราะห์และแนะนำ 1 ตัวเลือกที่เหมาะสมที่สุด:

**หลักการแนะนำ:**
- จำนวน < 1,000 ชิ้น → Offset (cost-effective ที่สุด)
- จำนวน 1,000–10,000 ชิ้น → Offset (optimal)
- จำนวน > 10,000 ชิ้น → พิจารณา Gravure ถ้างาน long-run
- งานบรรจุภัณฑ์ฟิล์ม → Gravure เสมอ
- งาน premium → Art Card + UV/Laminate
- งานประหยัด → Bond หรือ Art Card ธรรมดาไม่มี finishing

รูปแบบคำแนะนำ:
"**แนะนำ: [ตัวเลือก]** เพราะ [เหตุผล 2-3 ข้อ]
ต้นทุนต่อชิ้นต่ำที่สุด / คุณภาพเหมาะกับงาน / คุ้มค่าที่สุดสำหรับจำนวนนี้

ถ้า [เงื่อนไข] → ควรเปลี่ยนเป็น [ตัวเลือกอื่น]"

หลังคำแนะนำ เสนอ: "ต้องการใบเสนอราคาสำหรับตัวเลือกนี้? ใช้ /quote [spec งาน]"
